# Admin UI

Open `/admin` in the browser.

## What it shows
- Service status (DB path, retention, anchor status)
- Anchor log stream

## Pair iPhone
- Click "New pairing code" to generate a short-lived pairing QR.
- Scan from iPhone to sign in without manually typing the token.

## Anchor control
- Anchor is auto-started by default when the service starts.
- Use "Stop anchor" if you want to suspend Codex spawning.
